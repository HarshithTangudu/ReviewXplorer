# Technology Stack

## Frontend
- **Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling/UI:** Vanilla CSS (App.css, index.css)
- **Charts:** Recharts
- **Icons:** Lucide-React
- **HTTP Client:** Axios
- **Linting:** ESLint with TypeScript integration

## Backend
- **Framework:** FastAPI
- **Server:** Uvicorn
- **Language:** Python 3.x
- **Data Validation:** Pydantic

## Machine Learning / NLP
- **Library:** Hugging Face Transformers, PyTorch, Scikit-learn
- **Sentiment Model:** distilbert-base-uncased-finetuned-sst-2-english
- **Emotion Model:** bhadresh-savani/bert-base-uncased-emotion
- **Sarcasm Model:** jkandut/sarcasm-detection-bert

## Scraping & Data Collection
- **Web Automation:** Playwright (for dynamic sites like Amazon, Flipkart)
- **HTML Parsing:** BeautifulSoup4
- **API Clients:** PRAW (for Reddit)
- **Custom Downloader:** youtube-comment-downloader (for YouTube)
- **Data Processing:** Pandas, Numpy
